# Primer Pygame – Juego Breakout

## Descripción del proyecto
**Primer Pygame** es un proyecto de carácter **educativo y demostrativo**, basado en la recreación del clásico videojuego **Breakout**.  
El objetivo principal de este proyecto es servir como material de aprendizaje para el autor y para otras personas interesadas en el desarrollo de videojuegos utilizando Python.

Este proyecto **no tiene fines comerciales** y **no reclama derechos de autor** sobre la idea original del juego. Su desarrollo se realiza exclusivamente con fines educativos, prácticos y formativos.

---

## Tecnologías utilizadas
- **Lenguaje de programación:** Python  
- **Librería principal:** Pygame  

Pygame fue utilizada para la gestión de ventanas, eventos, gráficos, colisiones y control del ciclo principal del juego.

---

## Funcionalidades principales
- Sistema de escenas (menú, juego y fin del juego).
- Control de paleta mediante teclado.
- Sistema de colisiones entre bola, paleta y ladrillos.
- Conteo de puntaje y vidas.
- Pantalla de inicio y pantalla de término del juego.

## Ejecución del programa

### Opción 1: Ejecutar desde consola (modo desarrollo)
1. Asegúrese de tener **Python 3.x** instalado.
2. Instale la librería Pygame:
   ```bash
   pip install pygame
Ejecute el archivo principal:
python juego.py

Opción 2: Ejecutar mediante archivo ejecutable (Windows)

Ingrese a la carpeta dist/.

Ejecute directamente el archivo:
juego.exe
